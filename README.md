## About The Project

![](images/screenshot2.png)

Scrapaste is a scraping service that extracts information off darknet pastebins.

This project was originally created as a class project for my programming course

### Built With

<br>

- React.js
- Express
- Next.js
- Redux Toolkit
- MongoDB
- MUI
